| ID        | Review Question                                                                                        | Y/N |
| --------- | ------------------------------------------------------------------------------------------------------ | --- |
| **CI-1**  | Is the drawing number, title, revision, and sheet numbering correct and consistent?                    | ☐   |
| **CI-2**  | Is the correct drawing standard identified and applied (e.g., ASME Y14.5, ISO)?                        | ☐   |
| **CI-3**  | Are units clearly specified and used consistently throughout the drawing?                              | ☐   |
| **CI-4**  | Are all required views present and sufficient to fully define the part or assembly?                    | ☐   |
| **CI-5**  | Are section views, detail views, and auxiliary views used appropriately where needed?                  | ☐   |
| **CI-6**  | Are all features fully and unambiguously dimensioned?                                                  | ☐   |
| **CI-7**  | Are there no duplicate, conflicting, or missing dimensions?                                            | ☐   |
| **CI-8**  | Are datum features clearly identified and correctly established?                                       | ☐   |
| **CI-9**  | Are GD&T symbols correctly applied, complete, and in accordance with the stated standard?              | ☐   |
| **CI-10** | Are tolerances appropriate for the function, manufacturing process, and cost targets?                  | ☐   |
| **CI-11** | Are general tolerances specified and consistent with detailed tolerances?                              | ☐   |
| **CI-12** | Are hole callouts complete and correct (size, depth, thread, fit, quantity)?                           | ☐   |
| **CI-13** | Are fastener callouts correct and consistent with mating parts or assemblies?                          | ☐   |
| **CI-14** | Are surface finish requirements specified where functionally required?                                 | ☐   |
| **CI-15** | Are material specifications complete, correct, and unambiguous?                                        | ☐   |
| **CI-16** | Are heat treatment, coating, plating, or finishing requirements clearly defined?                       | ☐   |
| **CI-17** | Are notes clear, concise, and free of ambiguity or contradiction?                                      | ☐   |
| **CI-18** | Are manufacturing processes implied by the drawing reasonable and achievable?                          | ☐   |
| **CI-19** | Are critical-to-function features clearly identified or flagged?                                       | ☐   |
| **CI-20** | Are inspection and verification requirements clearly defined and measurable?                           | ☐   |
| **CI-21** | Are mass, balance, or center-of-gravity requirements specified if applicable?                          | ☐   |
| **CI-22** | For assemblies, are all parts correctly ballooned and listed in the BOM?                               | ☐   |
| **CI-23** | For assemblies, are mating relationships and interfaces clearly defined?                               | ☐   |
| **CI-24** | Does the drawing fully support fabrication, inspection, and acceptance without external clarification? | ☐   |
