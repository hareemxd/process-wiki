
# Problem Report

## Identification

| Field                  | Value                                                 |
| ---------------------- | ----------------------------------------------------- |
| **Problem Report ID**  | PR-YYYY-NNNN                                          |
| **Status**             | Open                                                  |
| **Date Submitted**     | YYYY-MM-DD                                            |
| **Submitted By**       | Name / Organization                                   |
| **Source**             | Internal / External                                   |
| **External Reference** | Customer Notice, Vendor PCN, Audit ID (if applicable) |


## Reason to Change

| Field               | Value                                                                       |
| ------------------- | --------------------------------------------------------------------------- |
| **Reason Category** | Corrective Action / Preventive Action / Opportunity                         |
| **Problem Summary** | One-paragraph factual description of the issue or opportunity               |
| **Trigger Event**   | Test failure, Obsolescence notice, Requirement change, Cost reduction, etc. |


## Initiating Item

| Field                            | Value                                          |
| -------------------------------- | ---------------------------------------------- |
| **Initiating Part Number (IPN)** | Part / Assembly / Document ID                  |
| **Item Type**                    | Hardware / Software / Firmware / Documentation |
| **Current Revision**             | As-Is revision at time of report               |
