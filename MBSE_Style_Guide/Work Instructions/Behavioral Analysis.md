# Procedure

## Create Use Cases

1. Identify System of Interest use cases.  

For each identified use case:  

2. Navigate to the **DRS Naval Electronics Profile > Templates**, then expand **Use Case, UCD, AD**.  
3. Right-click **Template UC > Copy**,right-click the **1 Use Cases** package in your model and **Paste**.  
4. Rename the Use Case `Use Case Name UC`, the nested diagram `Use Case Name UCD`, the nested activity `Use Case Name AD`, and the lower nested diagram `Use Case Name AD`.  
5. Open the Use Case Diagram and delete pre-existing notes.  
6. Write the pre-conditions that are required for the use case to begin.  
7. Populate the Use Case Description.  

### Layout Diagram

1. Populate the diagram title.  
2. From the **System Context (Structure)** package, drag the SoI (top center) and the appropriate actors (left) and the external systems (right) onto the diagram.  
3. Create **Associations** between the UC and the SoI, actors, and external systems.  
4. Use the **Align** tool in the top toolbar to resize and reposition the diagram elements to look organized and uniform.
